Goddies Lounge & Wine Bar

Project Handover & Operational Summary

This document provides a technical and operational overview of the Goddies Lounge Management System, including a breakdown of necessary infrastructure for long-term production use.

1. Technical Architecture

The application is a modern, high-performance web platform built with a dedicated frontend, serverless API layer, and real-time database capabilities.

  • Frontend: React.js with Vite and Tailwind CSS.
  • Backend: Vercel Serverless Functions (Node.js).
  • Database: Neon PostgreSQL (Primary Data).
  • Real-time: Firebase Realtime Database (Rider Tracking).
  • Auth: Firebase Multi-channel (Google, Phone OTP, Username).
